# Niro-Media-v1

![Niro Media Logo](src-tauri/icons/128x128.png)

**Niro Media** is a premium, high-performance desktop media player designed for a seamless playback experience. Built with Rust and React using the Tauri framework, it offers a secure and lightweight alternative to traditional media players.

## ✨ Features
- **Multi-Format Support**: Play videos (mp4, mkv, webm), audio (mp3, wav, flac), and view images.
- **Modern UI**: A sleek, customizable interface with multiple luxury themes.
- **Performance**: Powered by Rust for minimal resource usage.
- **Security**: Hardened with Content Security Policy (CSP) and secure local asset protocols.
- **Always on Top**: Optional mode for multitasking.
- **Playback Control**: Adjustable speed, loop playback, and intuitive keyboard shortcuts.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Rust (latest stable)
- Tauri CLI (`npm install -g @tauri-apps/cli`)

### Installation
1. Clone the repository (Access restricted to EstherMinds).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application in development:
   ```bash
   npm run tauri dev
   ```

### Building for Production
To create a production-ready installer:
```bash
npm run build
npm run tauri build
```
